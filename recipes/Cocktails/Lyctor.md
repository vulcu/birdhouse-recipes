# Lyctor

## Ingredients

| Qty.   | Item                |
| ------ | ------------------- |
| 3/4 oz | Absinthe            |
| 3/4 oz | Elderflower liqueur |
| 3/4 oz | Lillet Blanc        |
| 3/4 oz | Lemon juice         |
| 2-4 oz | Ginger beer         |
|        | Lemon twist         |

## Steps

1. Add rocks to highball glass, add and mix absinthe, elderflower
   liqueur, lillet blanc, and lemon juice.
1. Top up with ginger beer, to taste.
1. Garnish with lemon twist.
