# Old Fashioned

## Ingredients

| Qty.     | Item                   |
| -------- | ---------------------- |
| 1 oz     | Bourbon or Rye Whiskey |
| 1        | Sugar cube (1/2 tsp)   |
| 2 dashes | aromatic bitters       |
|          | Orange peel            |

## Steps

1. Muddle sugar and bitters.
1. Stir all over rocks.
1. Squeeze twist over drink, garnish.
