# Tequiliano

## Ingredients

| Qty.   | Item                    |
| ------ | ----------------------- |
| 1 oz   | Tequila blanco          |
| 1 oz   | Campari                 |
| 1 oz   | Sweet Vermouth          |
| 1/4 oz | Agave Nectar (optional) |
|        | Orange twist            |

## Steps

1. Stir all over rocks.
1. Squeeze twist over drink, garnish.
