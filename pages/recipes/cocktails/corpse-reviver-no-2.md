# Corpse Reviver No. 2

## Ingredients

| Qty.   | Item                                            |
| ------ | ----------------------------------------------- |
| 1/4 oz | Absinthe                                        |
| 1 oz   | Gin                                             |
| 1 oz   | Lillet Blanc, Cocchi Americano, or Dry Vermouth |
| 1 oz   | Orange liqueur                                  |
| 1 oz   | Lemon juice                                     |
|        | Orange peel                                     |

## Steps

1. Rinse cocktail glass with absinthe.
1. Add remaining, shake. Serve straight up.
1. Garnish with orange peel.
